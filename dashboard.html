Resumo do Projeto

Este projeto consistiu em uma página de login que realiza uma verificação com o backend utilizando requisições HTTP. A autenticação é feita via uma API desenvolvida em ASP.NET Core, e o frontend utiliza JavaScript para gerenciar a interação e o redirecionamento em caso de sucesso no login.
